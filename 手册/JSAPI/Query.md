# Query 查询

## 示例

Query 使用 SQL 构造查询，更多查询<a href="../进阶/复杂数据查询">查询数据</a> 文档。

```javascript
var query = new Query();

var result = query.Get({
  select: [":COUNT(id) as number"],
  wheres: [{ ":status": "状态", "=": 2 }],
  from: "foo_table",
});
```
