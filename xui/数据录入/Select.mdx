# Select 选择器

下拉选择器，为 html5 select 的功能增强型组件。

## 使用场景

- 弹出一个下拉菜单给用户选择操作，用于代替原生的选择器，或者需要一个更优雅的多选器时。

- 当选项少时（少于 5 项），建议直接将选项平铺，使用 Radio 是更好的选择。

## 示例

```json
{
  "Select": {
    "bind": "Select_value",
    "edit": {
      "type": "Select",
      "props": {
        "tree": true,
        "searchable": true,
        "xProps": {
          "$remote": {
            "process": "scripts.select.Tree",
            "query": {}
          }
        }
      }
    }
  }
}
```
