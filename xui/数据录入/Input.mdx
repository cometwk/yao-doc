# Input 输入框

通过鼠标或键盘输入内容，是最基础的表单域的包装。

## 使用场景

- 需要用户输入表单域内容时。

- 提供组合型输入框，带搜索的输入框，还可以进行大小选择。

## 示例

```json
{
	"Input": {
		"bind": "Input_value",
		"edit": {
			"type": "Input",
			"props": {}
		}
	}
}
```

## Props

```ts
interface IProps extends InputProps, Component.PropsEditComponent {}
```
