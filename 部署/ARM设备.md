## 连接设备

### 安装 ADB

ADB，全称 Android Debug Bridge，是 Android 的命令行调试工具。

[下载安装 ADB](https://developer.android.com/studio/releases/platform-tools)

### 常用命令

#### 设备列表

列出所有连接设备

```bash
adb devices
```

#### 上传文件

复制本地文件到设备

```bash
adb push <本地文件路径> <设备文件路径>
```

#### 下载文件

复制本地文件到设备

```bash
adb pull <设备文件路径> <本地文件路径>
```

#### 设备 shell

进入设备 Shell 环境

```bash
adb shell

```

## 安装 Yao

下载系统为 Linux 架构为 arm64 的制品，将其复制到 bin 目录。[下载地址](https://yaoapps.com/release)

使用 USB/Type-C 将设备连接至电脑

```bash
adb push yao-0.10.1-linux-arm64 /usr/bin/yao
```

在本地运行 adb shell, 进入设备进行配置

```bash
adb shell
```

在设备上运行，添加可执行权限

```bash
chmod +x /usr/bin/yao
yao version

```
