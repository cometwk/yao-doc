# InputNumber 数字输入框

通过鼠标或键盘，输入范围内的数值。

## 使用场景

- 当需要获取标准数值时。

- 提供格式化的数字输入，以此来约束输入内容。

## 示例

```json
{
	"InputNumber": {
		"bind": "InputNumber_value",
		"edit": {
			"props": {}
		}
	}
}
```
