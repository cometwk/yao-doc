# Text 标签

纯文本，仅用作展示。

## 使用场景

- 用户名

- 字段描述

## 示例

```json
{
	"Label": {
		"bind": ":name",
		"view": {
			"type": "Text",
			"props": {}
		}
	}
}
```

## Props

```ts
interface IProps {
	format?: string
}
```

## API

| 属性   | 说明 | 类型   | 默认值 | 必填 | 版本 |
| ------ | ---- | ------ | ------ | ---- | ---- |
| format | 格式 | string | -      | 否   |      |
